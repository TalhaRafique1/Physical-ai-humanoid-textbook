# Textbook Generation Backend

This is the backend API for the textbook generation system. It provides endpoints for generating, customizing, and exporting textbooks with AI assistance.

## Features
- Textbook generation with AI
- RAG (Retrieval-Augmented Generation) chatbot
- Authentication system
- Translation services
- Learning tools integration
- Export functionality

## API Endpoints
- `/api/docs` - API documentation
- `/api/textbook-generation` - Textbook generation endpoints
- `/api/rag` - RAG chatbot endpoints
- `/api/auth` - Authentication endpoints
- `/api/translation` - Translation endpoints
- `/api/learning-tools` - Learning tools endpoints
- `/health` - Health check endpoint
