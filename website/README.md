# AI-Native Interactive Textbook for Physical AI & Humanoid Robotics

This Docusaurus-based interactive textbook provides comprehensive coverage of Physical AI and Humanoid Robotics with integrated AI capabilities.

## Features

- **Interactive Learning**: Engaging content with quizzes, simulations, and interactive elements
- **AI-Powered Chatbot**: Integrated RAG chatbot for personalized learning assistance
- **Multilingual Support**: Translation capabilities including Urdu support
- **Comprehensive Coverage**: From Physical AI fundamentals to advanced humanoid robotics
- **Real-World Applications**: Industrial, healthcare, and domestic robotics applications
- **Future Perspectives**: Technological advances and societal implications

## Structure

The textbook is organized into the following main sections:

1. **Getting Started**: Introduction and setup guide
2. **Physical AI**: Fundamentals of sensing, perception, motion control, learning, and cognition
3. **Humanoid Robotics**: Bipedal locomotion, manipulation, and intelligence
4. **Applications**: Industrial, healthcare, and domestic applications
5. **Future Perspectives**: Technological advances and societal implications

## AI Integration

The textbook includes advanced AI features:

- **Chat Widget**: An AI assistant available on all pages to answer questions about the content
- **Translation Toggle**: Multilingual support for inclusive learning
- **Interactive Elements**: Simulations, exercises, and discussion forums

## Development

To run the textbook locally:

1. Install dependencies: `npm install`
2. Start development server: `npm start`
3. Build for production: `npm run build`
